# Excel Structure Configuration
# This file documents the expected Excel structure for reference

Sheet Name: Scenarios

Expected Row Labels (first column):
- Ops - Revenue
- Ops - EBITDA
- Ops - Payroll
- Ops - Margin
- Bridge - Revenue
- Bridge - EBITDA
- Bridge - Payroll
- Bridge - Margin
- Full - Revenue
- Full - EBITDA
- Full - Payroll
- Full - Margin

Expected Column Headers (columns B-F):
- FY25A
- FY26E
- FY27E
- FY28E
- FY29E

Data Format:
- Revenue: Integer values (e.g., 41699 = $41.699M in thousands)
- EBITDA: Integer values (e.g., 10299 = $10.299M in thousands)
- Payroll: Integer values (e.g., 19410 = $19.410M in thousands)
- Margin: Decimal values (e.g., 0.238 = 23.8%)

Notes:
- All financial values are in $000s (thousands)
- First column contains row labels
- Columns 2-6 contain the 5-year data
- Additional rows/columns can exist but won't be extracted
